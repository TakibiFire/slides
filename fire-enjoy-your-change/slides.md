# 人は今の人格が長続きすると考えがち

人が将来を想像する時**自然と「自分の性格や好みは変わらない」想定をする**

  * 就職、転職、結婚、ミッドライフ・クライシス、リタイア後の人生計画...
  * それかそもそもそんな想定すらしないかも？

<br>

でも<span font-bold text-purple>10年前を振り返ると自分が色々変わっていることに気づく</span>

* 性格、価値観、交友関係(友人・恋愛・仕事)が10年前と全く同じ人は少ない

# 実験内容

* 19,000人の幅広い年齢の成人を2グループに分ける
* 片方には「今後10年であなたの価値観がどれくらい変わるか」を予想してもらう
* もう片方には「ここ10年であなたの価値観はどのくらい変化したか」を聞く
* 同じことを「価値観」だけなく「性格」「好み」でも検証する。

その結果...

# <span text-red>1. 「自分はこういう人間だから」と決めつけるのをやめましょう</span>

まずはマインドセットが大事です。

変化は自然と起きます。新しい経験もどんどん取り込んでいきましょう。

<br>

# <span text-orange>2. 過去の成長を振り返りましょう</span>

過去を振り返り「自分は思ったより変化している」と気づけば、今の自分がまだ進行中だと気付けるでしょう。

---

# <span text-yellow>3. 新しい経験を求めましょう</span>

コンフォートゾーン (快適で慣れ親しんだ領域) に居続けると成長しません。

その外の快適ではない領域や新しい視点に触れ合うように挑戦しましょう。

成長好きな人たちと交流しましょう！

<br>

# <span text-green>4. デジタル・AIとの付き合い方に注意しましょう</span>

SNSやオススメ機能はあなたの好みを学習しそれに沿ったものを提供し続けますが、これはあなたの視野を狭め (エコーチェンバー)、将来の変化の可能性にも悪影響を及ぼします。

自分とは違った視点のコンテンツを積極的に求めていくと良いでしょう。

# <span text-blue>5.「今の状況はずっと続く」というマインドセットはやめましょう</span>

あなたの現在の人間関係も、好みも、困難でさえも、<br>
今後どんどん変わっていきます。

どんどん変わっていく前提でいると、ストレスが溜まりにくくなり、新しい変化にも飛び込みやすくなります。
