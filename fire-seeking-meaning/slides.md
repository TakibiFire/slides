# 再掲: 「人生における意味」・「意味の源」とは

* **人生における意味**:「この人生は生きるに値する」という実感
  * 将来の目標があり、自分の過去や現在に繋がりを感じ、<br>自分の人生には価値があると思えること
  * 見つけようと努力しないと見つからない
* **意味の源**: 「人生に意味を与えてくれるもの」 ([26種の意味の源](https://takibi-fire.com/slides/fire-nature-of-meaning/25))
  * 「笑いがない世界なんて死んだも同然だ」と思える人にとっての「笑い」
  * 「成長が感じられない世界なんて死んだも同然だ」と思える人にとっての「成長」
* **意味の源は複数持っておくと良い**

# 意味を見つける方法 1. 「やったことリスト」

[「第2章 バケットリストの欠点とリバースバケットリストの紹介」](https://takibi-fire.com/slides/fire-reverse-bucket-list/)でも紹介しました。

* **大小さまざまな「達成」を書き出す**
  * 大きな成果だけでなく、小さな成功体験や努力、誇りに思える瞬間も積極的に書き出す
  * 例：資格取得、昇進、プロジェクト成功、趣味の習得、目標達成、困難な状況の克服、
  * 新しいことに挑戦したこと、人から感謝されたことなど
* **具体的なエピソードを添える**
  * 単に「〇〇を達成した」と書くだけでなく、具体的なエピソードや、当時の感情、学びなどを加えることで、<br>
より深い振り返りができる
  * 例：マラソン完走 - 初めてフルマラソンに挑戦し、練習を重ねて完走できた時、達成感と自信に満ち溢れた。
* **カテゴリー別に分類する**
  * 年代別、人生のカテゴリー別（仕事、家族、趣味、人間関係など）に分類するとより多角的に自己分析できる

# 意味を見つける方法 2. 「なぜなぜ法」

* 「なぜなぜ法」: 子どもが聞くみたいになぜなぜと聞く <span text-5>([出典 (英語)](https://www.psychologytoday.com/us/blog/unnatural-selection/202101/5-tips-finding-meaning-in-regular-old-life))</span>
  * あなたが思ういい人とは？ なぜ？を最低2回聞く
  * いつ・どこにいるのが快適？　なぜ？なぜ？
  * あなたにとって理想の一日とは？ なぜ？なぜ？

* 結果をまとめて、数時間かけて自分にとって何が重要かを見出す

# 意味を見つける方法 5.

[第5章: 承認欲求から自由になる> 「私には価値がある」と思えない時は](https://takibi-fire.com/slides/fire-stop-seeking-approval/16)で<br>紹介した方法を再掲します

<fieldset border-gray border-2 rounded-4 bg-gray-500 bg-opacity-20 px-6 py-2
  w-fit m-auto mb-10 text-5.5>

* 「私には価値がある」と思えない時は **<span>「行動」「関与」</span>から始める**
  * 無理に「価値がある」と信じるのではなく、価値に繋がりそうな<br>行動から始めましょう
  * **他者への貢献や親切を実践する**: 小さな手助けや感謝を伝えるなど、<br><u>与える</u>行動をすると「自分は他者に良い影響を与えられる」という<br>感覚が育ちます
* 「自分に価値があるか」と考え続けるのではなく、<strong>「どうすれば<br>他者のために価値を生み出せるか」</strong>というふうに考えを切り替える

# 意味に関するよくある誤解

<div text-5.5>

* 意味の源は全ての人に共通？
  * 　<carbon-ArrowRight/>意味はめちゃくちゃ個人的です。人によって全然違うので自分で探しましょう
* 壮大な目的が必要？
  * <carbon-ArrowRight/>いいえ。日常的な小さな意味も同様に大事です。
* 意味は一生のテーマでないといけない？
  * <carbon-ArrowRight/>意味は時間とともに変化します
* 意味のある人生は幸せ？
  * <carbon-ArrowRight/>意味のある活動の中には感情面で苦しみを伴うものもあります
* 成功や達成が自動的に意味をもたらす？
  * <carbon-ArrowRight/>価値観との一致がより重要です。価値観に沿っていないと空虚さを感じます
</div>

# まとめ

* 人生における意味は、行動や自己探求を通じて**獲得**するものです
* 自分の「意味の源」のヒントを探しましょう
  * 過去の分析（やったことリスト、なぜなぜ法）
  * 極端な質問を考える（無かったら死ぬもの、墓石法）
* 最初から正解を求める必要はありません
  * 行動や人に働きかえたりすると、後から意味が見えてくることもあります<br>
    （自分の強みを活かす、小さな行動を積み重ねる）
* 意味は人それぞれ異なり、壮大である必要もありません。<br>自分だけの意味を焦らずに見つけていきましょう
