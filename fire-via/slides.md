# そもそも VIA とは

やり方を先に説明したので、VIA とは何かを説明します。

* VIA は "Values in Action（行動における価値）"の意味です。
* 人間の良い面、つまり"強み"に焦点を当てた心理学の研究から生まれました
* VIAでは、**あなたの考え方、感じ方、行動のうち、自分や他人にポジティブに働きかけるもの**を「強み特性」として認定し、全部で **<span text-green>24の「強み特性」</span>** を定義しています。

# 「強みを活かす」を実際にやってみて感じる幸福感

* 普段、自分が自然とできていることをあえて意識してやってみると、それが<br>自分の強みだと改めて認識できます

# 幸福度を感じやすい強みを鍛えていきましょう

例えば「愛情」の強みをトレーニングすれば、ポジティブな感情は増え、深い人間関係を築きやすくなります。
あなたの診断結果で「愛情」が上位か下位かは関係ありません。**強みは誰でも鍛えれば育ちます。**

* あなたは魔力を鍛えた結果、幸福も攻略できるようになるし、<br>魔法剣士になる <u decoration-wavy>(最強)</u>
