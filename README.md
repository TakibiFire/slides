# TakibiFire/slides

This repository contains slide presentations built with [Slidev](https://sli.dev/).

## Development

To start the development server:

```bash
pnpm run dev
```

## Build

To build the slides as a static web application:

```bash
pnpm run build
```

## Export

To export the slides to PDF, PPTX, or PNGs:

```bash
pnpm run export
