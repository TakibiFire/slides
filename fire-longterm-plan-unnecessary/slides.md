# 会社における短期目標と長期目標

* 短期目標 (~3ヶ月) の意義
  * 今季達成する目標を明確にする
  * 会社の業績、あなたの評価に直結
* **長期目標 (3~20年以上) の意義**
  * チーム・プロジェクトとしての方向性を決める
  * 全体として何を目指すかに一貫性を持たせる
  * 短期目標が行き当たりばったりにならないか確認できる
  * 長期目標が上手く定まっていたら、向かう方向は同じでも手段を変えることもできる

# 学び その1 - 長期目標はグループ向きのアイデア

* 会社では長期目標は確かに重要
  * でもそれは **たくさんの人に共通するお題目がないと一貫性が下がって<br>効率が悪いから**
* 個人には当てはまらないと今は思う
  * そもそも自分一人だから、他人との一貫性なんて必要ない
  * リタイア後、効率は求めていない

---

# 学び その3 - 長期目標を考えても<span text-red>あまり意味がない</span>

これは呪縛が解けてからさらに9ヶ月経った後の考えです。

* [自分の変化を楽しもう](https://takibi-fire.com/slides/fire-enjoy-your-change/)の回でも紹介したように、自分はゆっくりと変化していっている。
  * 特にリタイア以降、生活スタイルの好みも徐々に昔と違ってきている
* **長期目標なんか立てても、どうせ将来の自分とはズレ始める可能性大**

# 学び その4 - 長期目標が欲しかった理由は<span text-red>「不安」</span>

* **大海原に飛び出る時に、何も目標もないことを「やばい」と感じていた**
* でも今思うに、これは単に自分が作った不安
  * 今までの仕事の世界で大事にされていた考え方を引きずっていた

# まとめ

* 仕事生活に長期目標は大事でしたが、リタイア生活後はそれほど必要ではない話をしました
  * 長期目標がなく生きる感覚がないと不安な人も結構いるかもしれません

* むしろ逆に目標のない生活を一度経験してみてほしいです
  * 寄り道に真髄があります
