# 思い出の性質①：ピーク・エンドの法則

<fieldset border-orange border-2 rounded-4 bg-orange-500 bg-opacity-20 w-fit mx-auto my-6
  px-6 text-6.4 text-center>

思い出は「一番気持ちが動いた時」と「終わる時」の2つに支配される
</fieldset>

<div grid grid-cols-5 gap-6><div col-span-3>
<img src="/imgs/peak-end.png"/>
</div><div col-span-2 text-5.8>

楽しい1週間の旅行を想定してください。楽しさを覚えているのは最も楽しかったか辛かった時（ピーク）と最後の時（エンド）というものです。足して2で割った楽しさがその思い出となるという法則です。

ネガティブな悪い印象の場合にも当てはまります。
</div></div>

---

# ピーク・エンドの法則の例

* 遊園地でアトラクションの待ち時間が長かったり、途中で少し疲れたりしたとしても、一番楽しかった絶叫マシーンでの体験（ピーク）や、帰り際のパレード（エンド）の印象が強ければ、後から振り返ったときに「すごく楽しい一日だった！」と感じる
* レストランでの食事が全体的に美味しく、店員さんのサービスも良かったのに、最後の会計でもたついてしまったり、店を出る直前に嫌な出来事があったりすると、そのネガティブな終わりの印象が強く残り、食事全体の評価が下がる

---

# ピーク・エンドの法則の派生：「持続時間の無視」

<div w-fit m-auto>
出来事の印象がピークとエンドの2つだけに支配されるということは、<br>次の情報も含んでいます。
</div>

<fieldset border-orange border-2 rounded-4 bg-orange-500 bg-opacity-20 w-fit mx-auto my-6
  px-6 text-7.5 text-center>

出来事の途中にどれくらい全体的に楽しかった・嬉しかった<br>（または辛かった・嫌な気分になった）かという<br>「期間の長さ」「平均的な満足度」は<br>最終的な思い出に繋がらない
</fieldset>

<div w-fit m-auto>
例えば「だいたい楽しかった」旅行も、ピークやエンドが悪い印象だと<br>それで上書きされます。
</div>

# 思い出の性質②：強い思い出は予測が外れた時にできる

* 記憶の定着にはドーパミンが影響
* 脳は「自分の予測が外れた時に」ドーパミンをたくさん出す
  * 思いがけず祝われた、絶対負けると思った試合で勝った<br><carbon-ArrowRight/> 強い思い出(ピーク)になる
  * 祝われるとネタバレされた、絶対勝てると思った試合で勝った<br><carbon-ArrowRight/> 思い出にならない

<div m-auto flex>
<fieldset v-click=1 border-orange border-2 rounded-4 bg-orange-500 bg-opacity-20 w-fit m-6
  px-3 text-7.5 text-center>

ピークを作るには<br>「思いがけなさ」が必要
</fieldset>
<fieldset v-click=2 border-pink border-2 rounded-4 bg-pink-500 bg-opacity-20 w-fit m-6 pb-0
  px-3 text-7.5 text-center mt--1>
  <legend>重要</legend>
<div>「思いがけなさ」を自分で<br>演出するのは至難のわざ</div>
</fieldset>
</div>

# DIE WITH ZERO の間違い - その2

<fieldset border-purple border-2 rounded-4 bg-purple-500 bg-opacity-20 w-fit px-6 mx-auto
  text-7.5 text-center my-5>

「思い出に残るような経験」を目指すことがナンセンス
</fieldset>

* 「いい経験をすればいい思い出になる」という前提がそもそも間違い
* 強い思い出は「思いがけない」出来事が必要。自分で演出するのは難しい
  * いい思い出作りを期待すればするほど強い思い出にならない
* 思いがけない出来事がポジティブになる保証もない
* 長期の体験をしても思い出せるのは1,2個

# 補足: 経験すること自体は良い。動機が大事

* DIE WITH ZERO の言う「若いうちに経験をしよう」という結論自体はいいと思います。**それは経験をしないと経験を味わう力が伸びないからです**
  * 味わう力が増えれば日々の充実感が増します
* しかし「思い出を作るのは早めのほうが良いから」という理由で経験をするのは、<span text-red>今を楽しむ気持ちが減るので逆効果です</span>
* そしてこの話で言っている「経験」とは何かもう一度深く考えましょう。
  * 「友達と笑いあった思い出」にお金は全く関係ありません

# 補足: 脳は思い出すたびに思い出を書き換えている

2025年の研究

* 記憶は写真やビデオのように過去そのままを再生するものではなく、呼び出すごとに現在の状況に合わせて再構成される即興作品のようなものである
* 新たに作られた即興作品が再び記憶されることで、記憶が徐々に書き換わる
  * 思い出している「今」の影響を受ける
* 記憶はどんどん不確実な（自分に都合の良い）ものになっていく
