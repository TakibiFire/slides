# 思い出の性質①：ピーク・エンドの法則

<fieldset border-orange border-2 rounded-4 bg-orange-500 bg-opacity-20 w-fit mx-auto my-6
  px-6 text-6.4 text-center>

思い出は「一番気持ちが動いた時」と「終わる時」の2つに支配される
</fieldset>

<div v-click=1 grid grid-cols-5 gap-6><div col-span-3>
<img src="/imgs/peak-end.png"/>
</div><div col-span-2 text-5.8>

楽しい1週間の旅行を想定してください。楽しさを覚えているのは最も楽しかったか辛かった時（ピーク）と最後の時（エンド）というものです。足して2で割った楽しさがその思い出となるという法則です。

ネガティブな悪い印象の場合にも当てはまります。
</div></div>

---

# ピーク・エンドの法則の例

* 遊園地でアトラクションの待ち時間が長かったり、途中で少し疲れたりしたとしても、一番楽しかった絶叫マシーンでの体験（ピーク）や、帰り際のパレード（エンド）の印象が強ければ、後から振り返ったときに「すごく楽しい一日だった！」と感じる
* レストランでの食事が全体的に美味しく、店員さんのサービスも良かったのに、最後の会計でもたついてしまったり、店を出る直前に嫌な出来事があったりすると、そのネガティブな終わりの印象が強く残り、食事全体の評価が下がる

---

# ピーク・エンドの法則の派生：「持続時間の無視」

<div w-fit m-auto>
出来事の印象がピークとエンドの2つだけに支配されるということは、<br>次の情報も含んでいます。
</div>

<fieldset border-orange border-2 rounded-4 bg-orange-500 bg-opacity-20 w-fit mx-auto my-6
  px-6 text-7.5 text-center>

出来事の途中にどれくらい全体的に楽しかった・嬉しかった<br>（または辛かった・嫌な気分になった）かという<br>「期間の長さ」「平均的な満足度」は<br>最終的な思い出に繋がらない
</fieldset>

<div w-fit m-auto>
例えば「だいたい楽しかった旅行」「とても素晴らしかった演奏会」も、<br>エンドが悪い印象だと、全体の印象がそれで上書きされます。
</div>

# 思い出の性質③：「思い出作り」は変な方向性

<fieldset border-purple border-2 rounded-4 bg-purple-500 bg-opacity-20 w-fit px-6 mx-auto
  text-7.5 text-center my-5>

「思い出作り」としてやりそうな経験は<br>将来何度も味わえるような思い出にならない。
</fieldset>

* **自分にとって重要な幸せな思い出と「キラキラして見える幸せそうな経験」は全く方向性が違う**
* 人とのつながりや自分の努力から生まれる思い出は何度も味わえる
  * これは他者との深い関係や自己肯定感を繰り返し感じるから
* しかし<u decoration-pink decoration-wavy>お金で買える・インスタ映えしそうな</u>経験は興奮による短期的満足。<br>脳内で繰り返しても慣れていく。参考: [短期的満足に注意しよう](https://takibi-fire.com/slides/fire-hedonia/)

# これまでのまとめ：思い出の性質

* どんなに長く・たくさん経験しても印象に関係するのは1,2個
* いい思い出を意図的に作るのは難しい
  * 自分にとって「思いがけない」要素が必要
  * いい経験でも終わりが悪ければ悪い印象になる
* 「思い出作り」でやりたくなるキラキラしたものと、あなたの今の幸せな思い出はほぼ関係がない

# 思い出志向のリスク

「食事中、旅行中、映え写真を撮りまくる」行為は未来を想像してその未来から見た過去を楽しんでいる行為。つまり「記憶する自己」を重視している。

写真を撮らなければ、もっと今の瞬間に集中し楽しむことができる。

<fieldset border-purple border-2 rounded-4 bg-purple-500 bg-opacity-20 w-fit px-6 mx-auto
  text-7.5 text-center my-5>
  <legend>重要</legend>

「今ここを生きる」を「若いうちに経験に投資しよう」<br>とするのは全くの解釈間違い。

そもそも、未来の自分を喜ばそうとする「思い出作り」<br>という発想自体が「今ここを生きる」とは逆の考え方。
</fieldset>

# DIE WITH ZERO の間違い - その3

<fieldset border-purple border-2 rounded-4 bg-purple-500 bg-opacity-20 w-fit px-6 mx-auto
  text-7.5 text-center my-5>

「人生は経験の合計」ではない。<br>
人生のほぼ全ては記憶に残らないのだから、<br>記憶に残らない日々の満足度を上げるべき。
</fieldset>

---

# 補足 1. 経験すること自体は良い。動機が大事

* DIE WITH ZERO の言う「若いうちに経験をしよう」という結論自体はいいと思います。**それは経験をしないと経験を味わう力が伸びないからです**
  * 味わう力が増えれば日々の充実感が増します
* しかし「思い出を作るのは早めのほうが良いから」という理由で経験をするのは、<span text-red>今を楽しむ気持ちが減るので逆効果です</span>
* そしてこの話で言っている「経験」とは何かもう一度深く考えましょう。
  * 「友達と笑いあった思い出」にお金は全く関係ありません

---

# 補足 2. 思い出とうつ病との関連

* 「幸せな記憶を思い出す」のはほとんどの人にとってはポジティブな気持ちになりますが、うつ病の人には逆効果になることがあります（例: 今との比較をし始めたり）
* 「将来後悔しないために思い出作りする」という主張が DIE WITH ZERO にありますが、**高齢になると**ネガティブ感情やストレスを上手く処理できるようになり、**そもそも後悔しにくくなります。** <br>しかしうつ病の人はそうなりません。
