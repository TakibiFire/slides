# FI していても心が自立しているとは限らない

[Wikipedia](https://en.wikipedia.org/wiki/Financial_independence) での Financial Independence の定義の和訳はこうなっています。

> 「ファイナンシャル・インディペンデンス（経済的自立）」とは、個人または世帯が、**現在のライフスタイルを維持するために、積極的に働いて収入を得ることに依存しなくても生活費を賄えるだけの十分な財源を蓄積している状態**を指します。この財源は、投資資産、個人が利用する資産、不労所得、副業による収入、遺産、年金、その他のさまざまな収入源から成り立ちます。

ただ、せっかくお金に依存しなくても良くなった状態になったのに、<br>**<span text-red>「心がお金に依存している」状態の人は結構多いです</span>**。

# 原因1. ギリギリの FI

資産的にいつ FIRE できるかを示す資産額を **FIRE Number** といいます。
* 支出や現在のリスク・リターンを元に計算することが多いですが、<br>詳しいやり方はここでは紹介しません
* 「ここまで達成できれば FIRE できる」という目安を示してくれます。

**<span text-red>しかし大抵の場合、FIRE Number はギリギリを攻めている数字です。</span>**

* 一応リスクは込みで計算しますが、少しでも下振れると不安になる数字です
* 少し出費が増えたり、リターンが悪かったりするだけで焦りを感じがちです

---

# 原因2. 「ゼロで死ね」をFIREでやるのは無理に近い

* 余剰金を残さないように生きる = 超ギリギリを攻める生活
* 不安をいくつも抱えるスタイル
  * お金を早めに使わないといけないというマインド
  * 使いすぎると資金が枯渇する恐怖
  * そもそも何歳まで生きるかわからない不確実さ

「ゼロで死ね」のメッセージが印象的すぎるけど、本の主張はそこじゃない。

---

# 原因3. FIRE以前の習慣の継続

* **FIREに向かっているときの心はお金まみれ**
  * 資産のチェックを毎日する
  * 「FIRE Number まで後何円！ 到達まで後何日！」という気持ち
  * 資産家をフォローしたり資産運用の話を追いかけたり
  * 資産の上限で一喜一憂したり
* これらは<span text-10 text-red>強い習慣</span>となって残ります
  * 資産の上下は「ガチャ脳」を生み出す
    * 時々嬉しいと脳はハマる (参考: [習慣化の罠：依存・自己バイアス](https://takibi-fire.com/slides/fire-brain-addiction/))

# 原因3. FIRE以前の習慣の継続

# 原因4. 資産を語りたくなっちゃう・認められたくなっちゃう

FIREできる人は少ないです。あなたはすごい人です。

でもあなたの周りにFIREに理解がある人は少ないですよね。お金の話をするのもはばかられたり、働かないことを悪く思う人も多いです。

**そんなあなたを認めてくれる絶好の場所がSNSや動画です。**

"〇〇歳で〇〇円で FIRE！" を SNS で発信すれば人は集まってきます。<br>

**資産推移をポストして皆から「いいね」をもらうと、認められている感覚が得られるでしょう。**
これはハマるとそう簡単には抜け出せない脳内報酬の沼に陥ります。

# <span text-red>1. FIRE Number より下の数字 "Action Number" を探す</span>

現実的に考えて、FIRE Numberが分かるようなリテラシーのある人は、FIRE生活どころではない暴落が起きたら何かしらのアクション (復職だったり副業だったり)を起こします。

「将来の自分も何か行動を起こすだろうなぁ」と思えるのであれば、<span text-red>毎日一喜一憂するのは無駄に心をすり減らしていっている行為だと認識しましょう</span>。

必要なのは **「ここまで資産が下がったらアクションを起こすぞ」という数字、Action Number**です。例えば「(資産/支出)がここまで下がったら必ずアクションを起こす」という計画です。

# <span text-blue>5. FIREは一生の決断ではないことを意識する</span>

FIRE後に「未来永劫このライフスタイル」と思い込んでしまうと資産の変化に不安を感じてしまいます。

「FIREを決断できたんだから、暴落が来ても同じようになんとかできるでしょ」と思っておけば不安感もだいぶ減ります。

# まとめ

* 収入が心配なくなって経済的自立ができたなら、**次は心のFIを目指しましょう**
  * アクションを起こすべきラインを決めて、そのラインまで資産が<br>落ちないのであればお金への囚われを意識的に避けましょう
* **資産の上下、他人の資産情報、自分の資産自慢など、基本お金は<br>あなたの脳をつねにくすぐります**
  * FIRE後もお金に囚われる習慣を続けると、不必要に心をすり減らします。
  * 防御力を鍛えましょう。早めに鍛えたほうがお得です。
